🧩 조각 맞추기 자동 계산기

1. 맵 만들기 (7x7 그리드)

채우고 싶은 칸을 클릭해서 맵을 만들어주세요.

🔒 잠긴 영역: 한가운데 줄무늬 영역은 편집할 수 없지만 조각 배치는 가능합니다.

3. 자동 배치 계산

입력한 조각으로 맵을 최대한 채우는 방법을 찾아드립니다.

💡 우선순위 세트의 효과를 최대한 받도록 배치합니다.
애매한 칸 수일 때는 다른 세트를 사용해서라도 최대 저항을 달성합니다.

2. 보유 조각 입력

💡 안내: 사진 캡처는 정확성테스트 부족 주의. 100% 신뢰는 하지마세요. 하지만 입력이 더 귀찮을수도.

📷 사진에서 자동 인식

조각 이미지를 업로드하면 자동으로 모양을 분석해서 개수를 입력합니다. (여러 장 가능)

📖 이미지 업로드 사용법

1️⃣ 게임 화면 캡처

해당 경로로 이동해서 조각 보관함 화면을 캡처합니다.

2️⃣ 세트 효과 선택 및 캡처

2️⃣ 이미지 업로드

'📸 이미지 업로드' 버튼을 클릭하고 캡처한 이미지를 선택합니다.
여러 장의 이미지를 동시에 선택할 수 있습니다.

3️⃣ 자동 인식

업로드된 이미지에서 조각들을 자동으로 인식합니다.
조각의 모양과 등급(레어/에픽/슈퍼)이 자동으로 분석됩니다.

4️⃣ 세트 선택

인식된 조각들이 속할 세트를 선택합니다.
사진마다 다른 세트를 선택할 수 있습니다.

5️⃣ 입력 완료

인식된 조각들의 개수가 자동으로 입력됩니다.
필요하면 수동으로 수정할 수 있습니다.

6️⃣ 자동 배치 실행

맵을 만들고 세트 우선순위를 설정한 후
'🎯 자동 맞추기 시작' 버튼을 클릭하면 최적 배치를 계산합니다.

💡 팁

  • 조각들이 선명하게 보이도록 캡처해주세요
  • 한 화면에 여러 조각이 가로로 나열된 상태로 캡처하세요
  • 등급별로 다른 배경색(파란색=레어, 초록색=에픽, 빨간색=슈퍼)이 표시됩니다
  • 인식이 잘 안되면 수동으로 입력할 수 있습니다
