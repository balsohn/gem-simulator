# 💎 보석 가공 시뮬레이터

게임의 보석 가공 시스템을 재현한 웹 기반 시뮬레이터입니다. **레어 보석 세공**과 **에픽 보석 세공** 두 가지 모드를 제공하며, 전략적인 선택을 통해 최고 등급인 **슈퍼 에픽**을 달성하세요!

🌐 **플레이하기**: https://balsohn.github.io/gem-simulator/

## 🎮 게임 규칙 (레어 보석 세공)

### 기본 시스템
- **총 8번의 시도** 기회
- **17개의 슬롯**으로 구성된 보드 (0~16번)
- **시작 위치**: 0번 칸 (이동 후에는 0번으로 돌아갈 수 없음)
- **목표**: 15번 칸(슈퍼 에픽) 달성

### 슬롯 등급 (레어)
- **0~9번**: 희귀 (파란색)
- **10~12번**: 에픽 (보라색)
- **13~14번**: 희귀 (파란색)
- **15번**: 슈퍼 에픽 (빨간색) ⭐ **최고 등급!**
- **16번**: 꽝 (회색) - 게임 즉시 종료

### 가공 방법 (두 모드 공통)

#### 1️⃣ 세게 두드리기
- **이동 범위**: +3 ~ +6
- **사용 제한**: 무제한
- **특징**: 빠르게 앞으로 이동, 공격적인 전략

#### 2️⃣ 세공하기
- **이동 범위**: -3 ~ +2
- **사용 제한**: 3회
- **특징**: 뒤로도 이동 가능, 위치 조정에 유용

#### 3️⃣ 안정제 사용
- **이동 범위**: +0 ~ +4
- **사용 제한**: 3회
- **특징**: 안정적인 이동, 정밀한 착지

## ✨ 주요 기능

### 1. 실시간 게임 플레이
- 직관적인 UI로 쉬운 조작
- 실시간 위치 표시 및 등급 확인
- 가공 기록 로그
- 이동 숫자 이펙트 (±숫자 애니메이션)

### 2. 🤖 AI 기반 실시간 추천 시스템
- **동적 프로그래밍(DP) 알고리즘**으로 수학적 최적 전략 계산
- 현재 상태(위치, 턴, 자원)에 따른 실시간 추천
- 2,448가지 상태 분석 (17위치 × 9턴 × 4세공 × 4안정제)
- 각 행동의 성공 확률 계산 및 표시
- 페이지 로드 시 자동 계산 (~50ms 이내)
- DP 미완료 시 휴리스틱 폴백 제공

### 3. 📊 확률 통계 분석
- **10,000회 몬테카를로 시뮬레이션**
- 각 등급별 도달 확률 계산
- 무작위 전략 vs 최적 전략 비교

### 4. ⚙️ 상태별 최적 분석 (수동 어드바이저)
- 원하는 게임 상태(위치, 남은 턴, 세공/안정제 횟수)를 직접 입력
- 해당 상태에서 슈퍼 에픽 달성 확률이 가장 높은 최적의 행동을 즉시 분석 및 추천

### 5. 🎨 멋진 UI/UX
- 그라데이션 배경 및 애니메이션
- 반응형 디자인 (모바일 지원)
- 등급별 색상 구분
- 가공 시 애니메이션 효과
- 실시간 추천 배너 (자동 표시/숨김)

## 🚀 사용 방법

브라우저에서 `index.html` (레어 모드) 또는 `epic.html` (에픽 모드) 파일을 열어 플레이할 수 있습니다! 각 시뮬레이터 상단의 링크를 통해 모드를 전환할 수 있습니다.


## 🎓 전략 팁

### 초보자를 위한 팁
1. **0번에서 시작**: 첫 이동으로 3~6칸 앞으로
2. **처음에는 세게 두드리기**로 빠르게 목표 구간으로 이동
3. **목표 칸 근처**에 도착하면 안정제나 세공하기 사용
4. **슈퍼 에픽 칸 도달 후**: 계속 플레이 가능하지만 꽝 칸에 가면 게임 종료!
5. **남은 턴 수**를 항상 확인하며 플레이

### 고급 전략
1. **확률 활용**: 각 방법의 평균값 고려
   - 세게 두드리기: 평균 +4.5
   - 세공하기: 평균 -0.5
   - 안정제: 평균 +2
2. **위험 관리**: 꽝 칸에 가지 않도록 주의
3. **DP 기반 추천**을 적극 활용하여 최적의 플레이를 찾아보세요!
